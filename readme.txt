Add the IteroChat AI customer support chat widget to your WordPress site. Connect in one click, no code required.

== Description ==

IteroChat adds an AI customer support chat widget to your site. Click Connect, approve access on iterochat.com, choose your organization, and the widget goes live. No copying code snippets.

You manage the assistant (knowledge base, appearance, team handoff, conversations) from your IteroChat dashboard. This plugin links your site to your IteroChat organization and loads the widget.

== External services ==

This plugin connects to IteroChat, a third-party service, to authorize your site and load the chat widget.

* During connection, a new tab opens on iterochat.com so you can sign in and approve access. The plugin then exchanges an authorization request with the IteroChat API for an access token and your widget key. The data sent is your site host name and the authorization request details.
* On your public pages, the widget script is loaded from the IteroChat widget host, and your site visitors' chat messages are sent to IteroChat to be answered by the assistant.

Service: IteroChat. Terms of service: https://iterochat.com/terms . Privacy policy: https://iterochat.com/privacy

== Installation ==

1. Install and activate the plugin.
2. Open the IteroChat menu in your WordPress admin.
3. Click "Connect to IteroChat" and approve access for your organization on iterochat.com.
4. Return to WordPress. The chat widget is now live on your site.

== Frequently Asked Questions ==

= Do I need an IteroChat account? =
Yes. You can create one for free from the connect screen.

= Can I turn the widget off without disconnecting? =
Yes. Use the "Show the chat widget on this site" toggle on the plugin page.

= How do I disconnect? =
Click "Disconnect" on the plugin page, or open Settings then Connected apps in your IteroChat dashboard and disconnect the site there. Either way access is revoked right away.
